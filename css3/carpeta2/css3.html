box-shadow: el box-shadow es una propiedad que sirve para ponerle sombras a las cajas, ejemplo:



#id{
box-shadow: 2px 4px 15px 10px #000;
}

con los primeros pixeles que ponemos después del ";" son para indicar el "eje X" y los segundos pixeles son para indicar el "eje Y" de la caja, después marcamos los pixeles del tamaño del desenfoque, después cuanto borde va a tener la sombra (generalmente se le da 0), y después se indica el color de la sombra

#id{
box-shadow: -4px -8px 15px 5px rgb(255, 255, 255);
}


text-shadow: text-shadow también es para agregar sombras, pero en lugar de dárselas a las cajas, esta se las da a las palabras, ejemplo:


#id{
text-shadow: 2px 2px 7px #fff;
}

#id{
text-shadow: 2px 2px 7px #fff, 2px 2px 7px #fff;
}

outline: la propiedad outline es algo así como la propiedad "border" pero la diferencia es que outline si ocupa espacio de la caja a la que se le aplica, por ejemplo, si una caja tiene un "border-color: red;", la caja ocuparía espacio al interior de la misma, sin interrumpir a las demás cajas, asi que lo que hace outline es marcar fuera de la caja a la cual se le aplica pero tampoco interrumpe la posición de otras cajas, ejemplo:


#id{
background-color: #000;
height: 150px;
width: 150px;
margin: auto;
outline: 10px solid blue;
}


como se puede ver el input tiene un outline de 10px de color azul, el outline a diferencia del "border" no ocupan espacio, es decir que se pueden ver, pero al momento de chocar con otra caja, el outline solo le pasara por encima y no la moverá, ejemplo:




#id{
color: rgb(0, 0, 0);
background-color: rgb(251, 251, 251);
height: 150px;
width: 150px;
outline: 30px solid rgb(32, 31, 31);
}

position: position es una propiedad que sirve para posicionar un elemento usando: "top", "right", "bottom" y "left", ejemplo:

#id{
width: 160px;
height: 160px;
background-color: white;
position: static;
}
#id{
width: 100px;
height: 100px;
background-color: white;
position: relative;
top: -150px;
right: -140px;
}

IMPORTANTE: "position: static;" es para dejar el elemento en default, ya que no le hacemos ningún cambio, y con "position: relative;" se pueden usar el: "top", "right", "bottom" y "left"



IMPORTANTE: se pueden usar "top", "right", "bottom" y "left", pero hay que recordar que los 2 mas importante son "top" y "right", ya que si usamos bottom y top al mismo tiempo, al que tomara sin importar la posición sera al top, ya que es mas importante



IMPORTANTE: aunque modifiquemos la posición de algún elemento de HTML con la propiedad "position", el elemento al que lo cambiemos de posición, seguirá ocupando su lugar, solo que su caja estará en otra parte



z-index: z-index solo se puede utilizar si ponemos en un elemento "position: relative;" en CSS, y sirve para poner elementos encima de otros para cuando usemos "top" y "right", ejemplo:


#codigo-largo__z-index{
color: white;
background-color: black;
margin-bottom: 25px;
margin-top: 25px;
padding: 30px;
width: 400px;
height: 200px;
position: relative;
z-index: 50;
}
#id{
width: 100px;
height: 100px;
background-color: white;
position: relative;
top: -100px;
right: -160px;
}

IMPORTANTE: todo elemento con un "position: relative;" ya tiene un z-index, solo que este esta en 0, lo que significa que si a un elemento le ponemos "position: relative;", automáticamente se pondrá encima de cualquier elemento
